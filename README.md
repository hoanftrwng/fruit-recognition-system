# fruit-recognition-system
An end-to-end Computer Vision system for automated fruit recognition and classification using Machine Learning algorithms
Project Overview
This project addresses a Computer Vision challenge tailored for smart agriculture and automated retail sorting systems. The core of the system is an image processing pipeline capable of Feature Extraction and Classification to identify various fruit types from static images or real-time video streams. The project encompasses the entire lifecycle, from Data Preprocessing and Model Training to Inference execution.

Key Features
Data Pipeline: Implements robust image preprocessing techniques, including resizing, normalization, and color space conversion, to optimize input features for the classification model.
